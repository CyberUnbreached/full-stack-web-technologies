# Web Technologies Coursework

A collection of projects completed throughout a Web Technologies course that demonstrate the progression from a simple static website to a dynamic multi-page PHP web application.

The project was developed incrementally over the semester, with each version introducing new concepts and functionality. Earlier versions have been preserved to document the development process and showcase the evolution of the website.

---

## Overview

This repository contains eighteen versions of a web application created as part of a Web Technologies course.

Each version builds upon the previous one, gradually introducing:

* HTML5 page development
* CSS styling and layout improvements
* Navigation systems
* JavaScript interactivity
* Event handling
* Form processing
* PHP scripting
* Reusable server-side components
* Dynamic content generation
* Database-related coursework concepts

The final version represents the culmination of all assignments completed throughout the semester.

This repository serves as both:

1. A record of coursework completed during a Web Technologies course.
